| ADSL | Assymmetric Digital Subscriber Line | Linha de Assinatura Digital Assimétrica | Banda larga assimétrica via linha de assinante digital |
| BYOD | Bring Your Own Device | Traga seu próprio dispositivo | Significa o uso de qualquer dispositivo, de qualquer propriedade e em qualquer lugar. |
| Dial Up | Dial Up | Conexão Discada | Uma opção de baixo custo que usa qualquer linha telefônica e um modem, oferece baixa largura de banda. |
| DSL | Digital Subscriber Line | Linha de Assinatura Digital | Banda larga via linha de assinante digital |
| Extranet | Extranet | Extranet | É uma rede de computadores que permite acesso externo controlado, para negócios específicos ou propósitos educacionais. |
| Firewall | Firewall | Firewall | É um sistema de segurança de rede de computadores que restringe o tráfego da Internet |
| Hardware de Rede | Network Hardware | Hardware de Rede | É um conjunto de dispositivos físicos utilizados para conectar hosts e outros dispositivos em uma rede. |
| Host | Host | Host | É todo e qualquer dispositivo final conectado a uma rede e que participa diretamente da comunicação. Também é chamado de cliente. |
| Hotspot | Hotspot | Ponto de Acesso | É um ponto físico que permite acesso à Internet |

| Internetwork | Internetwork | Internetwork | Em sua forma abreviada "internet", é uma coleção mundial de redes interconectadas |
| Intranet | Intranet | Intranet | Se refere a uma conexão privada de LANs e WANs que pertence a uma organização. |
| IoT | Internet of Things | Internet das Coisas | É um rede de objetos físicos incorporados a sensores, software e outras tecnologias |
| IP | Internet Protocol | Protocolo de Internet | É um conjunto de regras para roteamento e endereçamento de pacotes de dados da rede |
| ISP’s | Internet Service Provider | Provedores de Serviços de Internet | São as empresas que fornecem acesso à Internet. |

| MAN | Metropolitan Area Network | Redes de Áreas Metropolitanas | É uma infraestrutura de rede que abrange uma área geográfica intermediária entre uma LAN e uma WAN. |

| SDSL | Symmetric Digital Subscriber Line | Linha de Assinatura Digital Simétrica | Banda larga simétrica via linha de assinante digital. |

| Software | Software | Software | É um conjunto de programas e drivers de dispositivos que permite a interação e troca de dados através da rede. |
| SOHO | Small Office-Home Office | Redes para pequenos escritórios e escritórios domésticos | Permite que computadores em um escritório em casa ou em um escritório remoto se conectem a uma rede corporativa, ou acessem recursos compartilhados centralizados. |
| Switch | Switch | Switch | Ele é responsável por conectar hosts, servidores e outros dispositivos na rede para que ocorra a comunicação e o compartilhamento de dados entre estes dispositivos. |

| WAN | Wide Area Network | Rede de Longa Distância | É uma infraestrutura de rede que abrange uma área geográfica extensa, como uma cidade, um país ou até mesmo o mundo todo. |

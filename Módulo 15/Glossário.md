

| Sigla / Palavra | Nome Completo em EN | Nome Completo em PT | Conceito |

| Aplicação P2P | Peer-to-Peer Application | Aplicação Ponto à Ponto | É um tipo de aplicação que permite que um dispositivo atue como cliente e servidor na mesma comunicação. |
| BitTorrent | BitTorrent | BitTorrent | É uma tecnologia ponto a ponto que permite que os usuários compartilhem partes de muitos arquivos entre si ao mesmo tempo. |
| BOOTP | Bootstrap Protocol | Protocolo de Bootstrap | Permite que uma estação de trabalho sem disco descubra seu próprio endereço IP, o IP de um servidor BOOTP na rede e um arquivo a ser carregado na memória para inicializar a máquina. |
| DHCPDISCOVER | DHCPDISCOVER | DHCPDISCOVER | É uma mensagem DHCPv4 que serve para identificar qualquer servidor DHCP disponível na rede. Em DHCPv6, é o comando SOLICIT que realiza esta função. |
| DHCPNAK | DHCPNAK | DHCPNAK | É uma mensagem DHCPv4 que serve para confirmar uma negativa DHCP, indicando que à oferta do servidor não é mais válida |
| DHCPOFFER | DHCPOFFER | DHCPOFFER | É uma mensagem DHCPv4 que responde à solicitação DHCPDISCOVER, oferecendo uma locação ao pedido do cliente. Em DHCPv6, é o comando ADVERTISE que realiza esta função. |
| DHCPACK | DHCPACK | DHCPACK | É uma mensagem DHCPv4 que serve para confirmar que a oferta do servidor ainda é válida. Em DHCPv6, é o comando REPLY que realiza esta função. |
| DHCPREQUEST | DHCPREQUEST | DHCPREQUEST | Este tipo de mensagem DHCPv4, serve para renovar uma locação de servidor antes do seu vencimento. Em DHCPv6, o comando INFORMATION REQUEST realiza esta função. |
| FQDNs | Fully Qualified Domain Names | Nomes de Domínio Totalmente Qualificados  | São endereços criados para converter um endereço numérico (IP) em um nome simples e reconhecível que identifica a localização de um serviço na internet. |
| GET | GET | GET | É uma solicitação de dados do cliente. Por exemplo, quando um cliente envia uma solicitação a um servidor web para requisitar páginas HTML. |
| GIF | Graphics Interchange Format  | Formato de Intercâmbio Gráfico | É um formato de imagem que pode ser usado para imagens estáticas ou animadas. |
| HTML | Hypertext Markup Language | Linguagem de Marcação de Hipertexto | É uma linguagem de marcação utilizada para estruturar páginas web.  |
| JPG | Joint Photographic Experts Group | Grupo de Peritos Fotográficos Conjuntos | Desenvolvido por um grupo de especialistas em fotografia, é um formato de compressão de imagem amplamente utilizado para armazenar e transmitir fotografias digitais.  |
| MKV | Matroska Video | Matroska Vídeo  | É um formato de vídeo que armazena várias faixas de áudio e legendas em um arquivo. |
| Modelo Cliente-Servidor | Client-Server Model | Modelo Cliente-Servidor | Define que o dispositivo que solicita as informações é chamado de cliente e o dispositivo que responde as solicitações é chamado de servidor.  |
| MOV | Quick Time | Quick Time | É um formato de armazenamento de arquivos multimídia desenvolvido pela Apple para ser usado no Apple QuickTime Player. |
| MPG | Motion Pictures Expert Group | Grupo de Especialistas em Cinema em Movimento | É um formato audiovisual elaborado pela Moving Picture Experts Group em 1991\. Ele comprime áudio em CD e vídeo bruto com qualidade VHS sem grandes perdas de qualidade.  |
| Nslookup | Nslookup | Nslookup | É um comando que permite consultar manualmente os servidores a partir dos nomes de domínio. Isso serve para descobrir o nome do host e o IP associado a estes domínios. |
| Período de Concessão | Period of Concession | Período de Concessão | É um período de tempo configurável em que o DHCP pode alocar endereços IP para dispositivos conectados em uma rede. |
| PNG | Portable Network Graphics | Gráficos de Rede Portáteis | É um formato de arquivo de imagem flexível, amplamente utilizado em sites para exibir imagens digitais de alta qualidade. |
| POST | POST | POST | É um tipo de mensagem que carrega arquivos de dados no servidor da web, como dados do formulário. |
| Protocolo Gnutella | Gnutella Protocol | Protocolo Gnutella | É um protocolo que permite que os usuários compartilhem arquivos completos com outros usuários. |
| PUT | PUT | PUT | É um tipo de mensagem que carrega recursos ou conteúdo para o servidor da web, como uma imagem. |
| SMB | Server Message Block | Bloco de Mensagens do Servidor | É um protocolo de compartilhamento de arquivos cliente/servidor, que descreve a estrutura de recursos de rede compartilhados, como diretórios, arquivos, impressoras e portas seriais. |
| SSL | Secure Sockets Layer | Camada de Soquetes Seguros | É um protocolo de segurança, utilizado para proteger uma conexão de internet criptografando os dados enviados do cliente para o servidor. |
| URI | Uniform Resource Identifier | Identificador Uniforme de Recursos | Serve para identificar um recurso da internet e o diferenciar de outros, através de um nome, local ou ambos. |
| URL | Uniform Resource Locator | Localizador Uniforme de Recursos | Serve para identificar um endereço da web ou o local de um recurso específico, sendo geralmente utilizado para identificar páginas na internet. |
